# Doemstika Course - Creative Coding

https://www.domestika.org/en/courses/3862-creative-coding-2-0-in-js-animation-sound-color

## Setup

```
$ mkdir sketches
$ cd sketches
$ npx canvas-sketch-cli sketch.js --new --open
```

## Run locally

```
$ npm install
$ npx canvas-sketch-cli <file_name> --open
```
